Read Me - Affine Cipher.

Run the script ./runaffine.sh to compile the program, and run the program 3 times with encryption data, decryption data and invalid keys (Covering all test cases).

For manual compilation
g++ -o affine main.cpp

For manual encryption
./affine -key B 3 -encrypt 4970512

For manual decryption
./affine -key B 3 -decrypt 54A320B
